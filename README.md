# `sellaut`

[Cellular automata](https://en.wikipedia.org/wiki/Cellular_automaton) in the [CLI](https://en.wikipedia.org/wiki/Command-line_interface).

## Stack

* Script: [Python](https://www.python.org/), [Rust](https://rust-lang.org/), [C++](https://en.wikipedia.org/wiki/C%2B%2B)

## Usage

```console
$ git clone https://github.com/gongahkia/sellaut & cd sellaut

$ cd cpp/lngtant && make run
$ cd cpp/elmrule && make run
$ cd cpp/cycaut && make run

$ cd python/brbr/src && python3.11 main.py
$ cd python/frfrmd/src && python3.11 main.py
$ cd python/prtsim/src && python3.11 main.py
$ cd python/trgn/src && python3.11 main.py

$ cd rust/cnwgol && cargo run
$ cd rust/wrwrld/src && cargo run
```
